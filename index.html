Basic structure of HTML:

The HTML is based liked a body of a human, countaining two things:

  • 1. Head: In the head we have the meta configuration, or, as we like to say, the way we're going to start;
  • 2. Body: The body is the place where we show what we have, and that's the place where we put our effort.


Talking about TAGS:

Let's review some of the foundational TAGS below:

  1. Strong: 
    Use the TAG: <b> </b>
  2. Italic: 
    Use the TAG: <i> </i>
  3. Header:
    Use the TAG: <h1> </h1> to <h6> </h6>

  4. Paragraph:

    Use the TAG: <p> </p>
  5. Mark: 
    Use the TAG: <mark> </mark>
  6. Small 
    Use the TAG: <small> </small>
  7. Underline 
    Use the TAG: <u> </u>
  8. Line 
    Use the TAG: <hr> </hr>
  9. Font 
    Use the TAG: <font> </font>. The font tag was used in HTML 4 to specify the font face, font size, and color of text. Not Supported in HTML5.
  10. Del 
    Use the TAG: <del> </del>
  11. Abbr 
    Use the TAG: <abbr title=""> </abbr>
